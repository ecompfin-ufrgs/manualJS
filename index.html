Introdução

Este trabalho objetiva apresentar a linguagem JavaScript. Sua escrita pretende-se dar de modo iterativo e incremental. Portanto, este manual não se apresenta de forma acabada, mas em constante evolução.

Observe que uma linguagem (humana ou de computador) é um construto usado para um indivíduo - chamado emissor - envie sinais a outro indivíduo - chamado receptor. Ao conjunto de sinais ou símbolos disponíveis em uma linguagem chama-se léxico. O emissor transmite sinais, ao rec objetivando comunicar uma mensagem ao receptor.

A fim de que a mensagem possua sentido é ncessário que os sinais estejam organizados segundo uma regra denominada sintaxe. A sintaxe prescreve a forma como os símbolos léxicos devem estar ordenados para formar termos que designam objetos do mundo (concreto ou abstrato) conhecidos tanto pelo emissor quanto pelo receptor. Ela também determina como os termos devem ser dispostos para formar mensagens providas de significado ou sentido. Chamamos ao conjunto destes objetos e de proposições com sentido de semântica da linguagem.

Ou seja, uma linguagem é composta pelos três seguintes elementos:

  • semântica;
  • léxico, e
  • sintaxe.

O léxico da linguagem JavaScript é formado pelo conjunto de caracteres UTF-8 e, por isso, não nos alongaremos na sua exposição. Na páginas seguintes, focaremos nossa atenção na apresentção da semântica da linguagem, isto é, o conjunto de objetos da linguagem acompanhado das operações que podem ser realizadas com eles.

A implementação da linguagem JavaScript

A fim de tornar a apresentação mais prática e fixar melhor o conteúdo, usaremos o aprendido para escrever pequenos programas em avaScript de modo a obter a fluência necessária ao seu emprego real. Para isso, necessitamos de escolher uma implementação JavaScript para executar nossos programas. Ocorre que JavaScript possui diversas implementações, destacando-se, entre estas, aquelas incorporadas aos navegadores Web, já que estes utilizam, por padrão, JavaScript como sua linguagem de script nativa. Além destas, o Node.JS é uma outra implementação do JavaScript particularmente importante, uma vez que é a primeira a permitir a execução do código diretamente no sistema operacional do computador onde está instalada.

Naturalmente, diferentes tempos de execução oferecem diferentes API para operar os objetos que têm acesso. Assim, o Node.JS acessa objetos da memória livremente ao passo que os navegadores são limitados pela restrição de segurança implementada por eles para evitar ataques via Internet.

Como os objetos componentes dos computadores são mais conhecidos dos que os objetos da rede Web, optamos por inicialmente empregar o Node.JS em nossos exemplos. Posteriormente, avançaremos para o uso da API Web dos navegadores, permitindo a construção de interfaces gráficas de usuário a partir da integração com elementos HTML estilizados por seletores CSS.
