Introdução

String é o tipo de dado usado para armazenar textos. Formalmente, diz-se que um string é uma cadeia (sequência) de caracteres.

As operações que podem ser realizadas com strings são:

  1. String
  2. Number
  3. Bigint
  4. Boolean
  5. Undefined
  6. Null
  7. Symbol
  8. Object
