Introdução

Linguagens de programação de alto nível manipulam objetos abstratos construídos na própria linguagem que representam endereços de memória de objetos concretos do hardware, tais como, a própria memória principal ou dispositivos de entrada e saída de dados. Estes endereços de memória associados ao conjunto de operações que podem ser realizados com os dados armazenados neles formam o que se denomina tipo de dado.

JavaScript possui sete tipos de dados primitivos, quais sejam:

  1. String
  2. Number
  3. Bigint
  4. Boolean
  5. Undefined
  6. Null
  7. Symbol

Além destes tipos, existe o tipo object . Entre os tipos Object mais importantes, tem-se Function, Array, Date, RegExp e Error.

Cada um dos sete tipos de dados e as operações passíveis de serem realizadas com eles é apresentado em outras páginas deste manual. Para isso, apresentaremos inicialmente a forma de se declarar objetos em JavaScript.

Variável

Os objetos em memória são acessados por meio de um identificador denominado variável. Cada variável tem o tipo determinado pelo objeto, sendo ela apenas uma etiqueta para designar o objeto em questão.

Para acessar um objeto, utiliza-se a variável que o identifica. Por isso, uma boa prática de programação é declarar a variável antes de utilizá-la pela primeira vez, explicitando o tipo de dado que a variável representa.

Declaração de variáveis

Exsitem três formas de declarar uma variável em JavaScript. A sintaxe a seguir serve para escopo global, permitindo, inclusive, a inicialização do valor. 

                
                    var nome_variavel = valor_variavel;

Observe que, por padrão, não se faz anotação de tipos em JavaScript, porém está sendo discutida a inclusão de sintaxe na linguagem para permitir que os tipos possam ser especificados no código. Confira EcmaScript proposal: type annotations .

Quando a variável é de escopo local, utiliza-se a sintaxe seguinte: 

                
                        let nome_variavel = valor_variavel;

Por fim, quando se deseja declarar uma constante, usa-se o que segue: 

                
                    const nome_variavel : tipo_de_dado = valor_variavel;

Exemplo 1: declarando um string constante

                
                    const nome = "Isto é um texto";

Observe que strings precisam vir dentro de aspas duplas ou simples.

Exemplo 2: declarando um number local

                
                    let idade = 8;
